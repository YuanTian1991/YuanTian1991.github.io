UCL Citrix Client Install on Linux

Firstly, I can only use version 13.8, instead of 13.10, which is vital and this information was searched from website. If you install latest version, the is an error like Cannot connect to "0.0.0.2-Desktop Corporativo"

From the official website, download the 13.8.0 version Receiver linux (X86_64)

Then use commend to install:

sudo dpkg -i icaclient_13.8.0.10299729_amd64.deb

You may encounter error like missing libwebkitgtk .e.g, try fix with below command:

sudo apt-get install libwebkitgtk-dev

Then, still it's not working, with a new error like "You have not chosen to trust ...", Then I found another solution tip from below website, so use below command:

sudo ln -s /usr/share/ca-certificates/mozilla/* /opt/Citrix/ICAClient/keystore/cacerts

Now it has been successfully installed. When using it, open UCL's web desktop, login, click the icon computer, and download one .ica file, click the file, then the remote virtual desktop would be opened.
